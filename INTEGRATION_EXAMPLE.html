Amtrak Live - Integration Example

This example shows how to integrate Amtrak Live data from GitHub Pages into your own website.

Loading train data...

Integration Code

To integrate this into your own website, use this JavaScript code:

// Set your GitHub Pages URL
const API_BASE = 'https://YOUR_USERNAME.github.io/YOUR_REPO_NAME';

// Fetch train data
fetch(`${API_BASE}/api/trains.json`)
  .then(res => res.json())
  .then(trains => {
    trains.forEach(train => {
      // Use the image URLs directly
      const backgroundImg = `${API_BASE}/${train.background_img}`;
      const foregroundImg = `${API_BASE}/${train.foreground_img}`;
      
      // Your custom rendering code here
      console.log(`Train ${train.id}:`, {
        backgroundImg,
        foregroundImg,
        animationStart: train.animation_start
      });
    });
  })
  .catch(error => console.error('Error:', error));
